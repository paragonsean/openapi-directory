

# JsonArtist

This class represents an artist. An artist is a musician or a group of musicians. Each artist has a definite <a href=\"http://wiki.musicbrainz.org/MBID\">Musicbrainz Identifier</a> (MBID) with which the artist can be uniquely identified.

|**tmid** | **BigDecimal** | unique Ticket Master Identifier (TMID), e.g. &lt;em&gt;735610&lt;/em&gt; |  [optional] |
