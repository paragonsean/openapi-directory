

# VisualElementChromaKey

Allows to define a color (or a range of colors) that will be converted to transparent

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**color** | **String** | Set the color for which alpha will be set to 0 (full transparency) |  |
|**tolerance** | **Integer** | Makes the selection more or less sensitive to changes in color. A value of 1 will select only the provided color. A value of 100 will select all colors, so the full canvas |  [optional] |
