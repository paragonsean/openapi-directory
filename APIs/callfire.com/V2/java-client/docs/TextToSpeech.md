

# TextToSpeech

Request object is used to create a sound from provided text using text to speech engine

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**message** | **String** | A text to be turned into sound |  [optional] |
|**voice** | [**VoiceEnum**](#VoiceEnum) | A voice to be used. Available values: MALE1, FEMALE1 , FEMALE2, SPANISH1, FRENCHCANADIAN1 |  [optional] |
