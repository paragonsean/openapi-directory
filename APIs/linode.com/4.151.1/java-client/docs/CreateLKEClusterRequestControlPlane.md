

# CreateLKEClusterRequestControlPlane

Defines settings for the Kubernetes Control Plane. Allows for the enabling of High Availability (HA) for Control Plane Components. Enabling High Availability for LKE is an **irreversible** change. 

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**highAvailability** | **Boolean** | Defines whether High Availability is enabled for the Control Plane Components of the cluster. Defaults to &#x60;false&#x60;.  |  [optional] |
