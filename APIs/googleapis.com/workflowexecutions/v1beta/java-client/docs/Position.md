

# Position

Position contains source position information about the stack trace element such as line number, column number and length of the code block in bytes.

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**column** | **String** | The source code column position (of the line) the current instruction was generated from. |  [optional] |
|**length** | **String** | The number of bytes of source code making up this stack trace element. |  [optional] |
|**line** | **String** | The source code line number the current instruction was generated from. |  [optional] |
