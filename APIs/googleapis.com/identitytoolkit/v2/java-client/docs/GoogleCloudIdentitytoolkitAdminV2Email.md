

# GoogleCloudIdentitytoolkitAdminV2Email

Configuration options related to authenticating a user by their email address.

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**enabled** | **Boolean** | Whether email auth is enabled for the project or not. |  [optional] |
|**passwordRequired** | **Boolean** | Whether a password is required for email auth or not. If true, both an email and password must be provided to sign in. If false, a user may sign in via either email/password or email link. |  [optional] |
