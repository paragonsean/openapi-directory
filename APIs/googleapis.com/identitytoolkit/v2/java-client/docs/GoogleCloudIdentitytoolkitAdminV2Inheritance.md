

# GoogleCloudIdentitytoolkitAdminV2Inheritance

Settings that the tenants will inherit from project level.

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**emailSendingConfig** | **Boolean** | Whether to allow the tenant to inherit custom domains, email templates, and custom SMTP settings. If true, email sent from tenant will follow the project level email sending configurations. If false (by default), emails will go with the default settings with no customizations. |  [optional] |
