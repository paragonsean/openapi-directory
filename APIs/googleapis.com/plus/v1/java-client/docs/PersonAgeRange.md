

# PersonAgeRange

The age range of the person. Valid ranges are 17 or younger, 18 to 20, and 21 or older. Age is determined from the user's birthday using Western age reckoning.

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**max** | **Integer** | The age range&#39;s upper bound, if any. Possible values include, but are not limited to, the following:   - \&quot;17\&quot; - for age 17  - \&quot;20\&quot; - for age 20 |  [optional] |
|**min** | **Integer** | The age range&#39;s lower bound, if any. Possible values include, but are not limited to, the following:   - \&quot;21\&quot; - for age 21  - \&quot;18\&quot; - for age 18 |  [optional] |
