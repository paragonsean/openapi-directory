

# ImageAnnotationContext

If an image was produced from a file (e.g. a PDF), this message gives information about the source of that image.

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**pageNumber** | **Integer** | If the file was a PDF or TIFF, this field gives the page number within the file used to produce the image. |  [optional] |
|**uri** | **String** | The URI of the file used to produce the image. |  [optional] |
