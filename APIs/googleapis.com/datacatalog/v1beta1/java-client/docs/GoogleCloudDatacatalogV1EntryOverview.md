

# GoogleCloudDatacatalogV1EntryOverview

Entry overview fields for rich text descriptions of entries.

| Name | Type | Description | Notes |

|**overview** | **String** | Entry overview with support for rich text. The overview must only contain Unicode characters, and should be formatted using HTML. The maximum length is 10 MiB as this value holds HTML descriptions including encoded images. The maximum length of the text without images is 100 KiB. |  [optional] |
