

# GalleryReference

|**host** | **String** | The name of the host for the community gallery template. |  [optional] |
|**isModified** | **Boolean** | If a user has manually edited the community gallery template. |  [optional] |
|**owner** | **String** | The name of the owner for the community gallery template. |  [optional] |
|**repository** | **String** | The name of the repository for the community gallery template. |  [optional] |
|**signature** | **String** | The signature of the community gallery template as computed at import time. This value is recomputed whenever the template is updated from the gallery. |  [optional] |
|**version** | **String** | The version of the community gallery template. |  [optional] |
