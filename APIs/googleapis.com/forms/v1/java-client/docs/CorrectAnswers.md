

# CorrectAnswers

The answer key for a question.

## Properties

|**answers** | [**List&lt;CorrectAnswer&gt;**](CorrectAnswer.md) | A list of correct answers. A quiz response can be automatically graded based on these answers. For single-valued questions, a response is marked correct if it matches any value in this list (in other words, multiple correct answers are possible). For multiple-valued (&#x60;CHECKBOX&#x60;) questions, a response is marked correct if it contains exactly the values in this list. |  [optional] |
