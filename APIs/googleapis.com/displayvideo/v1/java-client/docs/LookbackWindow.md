

# LookbackWindow

Specifies how many days into the past to look when determining whether to record a conversion.

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**clickDays** | **Integer** | Lookback window, in days, from the last time a given user clicked on one of your ads. |  [optional] |
|**impressionDays** | **Integer** | Lookback window, in days, from the last time a given user viewed one of your ads. |  [optional] |
