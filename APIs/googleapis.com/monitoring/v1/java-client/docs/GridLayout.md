

# GridLayout

A basic layout divides the available space into vertical columns of equal width and arranges a list of widgets using a row-first strategy.

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**columns** | **String** | The number of columns into which the view&#39;s width is divided. If omitted or set to zero, a system default will be used while rendering. |  [optional] |
|**widgets** | [**List&lt;Widget&gt;**](Widget.md) | The informational elements that are arranged into the columns row-first. |  [optional] |
