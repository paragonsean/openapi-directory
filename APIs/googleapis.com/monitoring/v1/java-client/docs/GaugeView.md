

# GaugeView

A gauge chart shows where the current value sits within a pre-defined range. The upper and lower bounds should define the possible range of values for the scorecard's query (inclusive).

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**lowerBound** | **Double** | The lower bound for this gauge chart. The value of the chart should always be greater than or equal to this. |  [optional] |
|**upperBound** | **Double** | The upper bound for this gauge chart. The value of the chart should always be less than or equal to this. |  [optional] |
