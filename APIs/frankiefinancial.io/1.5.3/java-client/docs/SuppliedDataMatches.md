

# SuppliedDataMatches

The results of the comparison of the supplied data (above) to that found on file with the ABR. If the value is missing, then the comparison was not run. There will likely be an issue highlighted in the issues_list 

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**matchedAcn** | **Boolean** | Did the supplied ACN match the ACN on file with the ABR? Only truly relevant if ABN is supplied as well.  |  |
|**matchedCompanyType** | **Boolean** | Did the supplied company type match the company type on file with the ABR?  |  |
|**matchedName** | **Boolean** | Did the supplied name match (or closely match) the name on file with the ABR  |  |
