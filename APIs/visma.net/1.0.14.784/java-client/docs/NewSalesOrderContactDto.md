

# NewSalesOrderContactDto

Defines a new contact when creating a new sales order.  When the contact information is provided, it is expected to provide all needed fields for the contact(name, attention etc).  No default values from the customer will be set for the non defined contact fields.

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**attention** | **String** | Sets any attention for the contact |  [optional] |
|**email** | **String** | Sets the e-mail address of the contact |  [optional] |
|**name** | **String** | Sets the full name of the contact |  [optional] |
|**phone1** | **String** | Sets the primary phone number of the contact |  [optional] |
