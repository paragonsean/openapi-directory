

# FolderInfo

Information about a image store folder. It includes how many files this folder contains and its image store relative path.

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**fileCount** | **UUID** | The number of files from within the image store folder. |  [optional] |
|**storeRelativePath** | **String** | The remote location within image store. This path is relative to the image store root. |  [optional] |
