

# LoyaltyAccountMapping

Represents the mapping that associates a loyalty account with a buyer.   Currently, a loyalty account can only be mapped to a buyer by phone number. For more information, see  [Loyalty Overview](https://developer.squareup.com/docs/loyalty/overview).

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**createdAt** | **String** | The timestamp when the mapping was created, in RFC 3339 format. |  [optional] |
|**id** | **String** | The Square-assigned ID of the mapping. |  [optional] |
|**phoneNumber** | **String** | The phone number of the buyer, in E.164 format. For example, \&quot;+14155551111\&quot;. |  [optional] |
