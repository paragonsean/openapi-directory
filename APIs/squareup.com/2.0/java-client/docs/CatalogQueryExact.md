

# CatalogQueryExact

The query filter to return the search result by exact match of the specified attribute name and value.

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**attributeName** | **String** | The name of the attribute to be searched. Matching of the attribute name is exact. |  |
|**attributeValue** | **String** | The desired value of the search attribute. Matching of the attribute value is case insensitive and can be partial. For example, if a specified value of \&quot;sma\&quot;, objects with the named attribute value of \&quot;Small\&quot;, \&quot;small\&quot; are both matched. |  |
