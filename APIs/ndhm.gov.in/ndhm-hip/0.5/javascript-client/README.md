# health_repository_provider_specifications_for_hip

HealthRepositoryProviderSpecificationsForHip - JavaScript client for health_repository_provider_specifications_for_hip
The following are the specifications for the APIs to be implemented at the Health Repository end if an entity is only serving the role of a HIP.
