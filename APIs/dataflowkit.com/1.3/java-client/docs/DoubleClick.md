

# DoubleClick

Double clicks on a target element (such as a link, button, checkbox, or radio button) with specified CSS Selector.

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**ignoreIfNotPresent** | **Boolean** | This optional parameter is useful when the target element occasionally may not be present in the DOM. |  [optional] |
|**selector** | **String** | Must be a valid CSS Selector |  [optional] |
|**skipLastIteration** | **Boolean** | It is only used for click action inside a loop only. Skips the last iteration. |  [optional] |
